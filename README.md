# RanneXu.github.io

Personal academic homepage for Ziran Xu.

## How to publish

1. Upload `index.html`, `publications.html`, and the `assets/` folder to the root of this repository.
2. In GitHub, go to **Settings → Pages**.
3. Set the source to **Deploy from a branch**, branch `main`, folder `/root`.
4. Wait for GitHub Pages to finish deployment.

The site should then be available at:

```text
https://RanneXu.github.io
```

## What to edit first

- Replace the portrait placeholder in `index.html` with your own photo.
- Replace `ziranxu@ucsb.edu` if you prefer another email.
- Replace the placeholder Google Scholar and CV links.
- Update `publications.html` with real papers, preprints, posters, or research notes.
